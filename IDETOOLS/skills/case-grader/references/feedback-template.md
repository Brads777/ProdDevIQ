Format all section titles as **Heading 3** (bold).

### Overall Assessment

Write 2-4 sentences of general evaluation. Match tone to score:
- 90+ : May use "excellent" or "outstanding"
- 75-89: May use "solid", "competent", "good"
- 60-74: Use "adequate", "developing", "shows potential"
- Below 60: Use "needs significant improvement", "incomplete"

---

### Areas for Improvement

List specific areas following rubric order:
- [Area 1]
- [Area 2]
- Use mandatory feedback phrases verbatim where triggered

---

### Recommendations for Your Career

Write 1-2 sentences summarizing improvement priorities based on potential career impact.
Connect the feedback to real-world consulting and management scenarios.

---

## Pre-Delivery Checklist

Before finalizing feedback, verify each item:

- [ ] Total points sum correctly to the score shown
- [ ] FROM line checked (2 points)
- [ ] CLV marked "Not Applicable" with 2/2 (unless case involves customer valuation)
- [ ] Exhibits identified by CONTENT KEYWORDS, not by letter label
- [ ] Extra/reordered exhibits did not cause subsequent frameworks to be missed
- [ ] Personal opinions only flagged if clearly stated ("I believe," "I think," "in my opinion")
- [ ] Exact sentences quoted when flagging opinion issues
- [ ] Mandatory feedback phrases used verbatim where triggered
- [ ] Tone matches score level (no "excellent" below 90, no "very good" below 75)
- [ ] All section titles formatted as Heading 3
- [ ] Cover page penalty applied if present (20% deduction)
- [ ] Output saved as .docx to /mnt/user-data/outputs/
