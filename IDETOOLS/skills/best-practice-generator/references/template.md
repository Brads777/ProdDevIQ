# Best Practice Case Analysis Template

## Memo Header Format
```
[Case Name]

TO:     [Recipient Name and Title]
FROM:   [Your Name and Title]  ← CRITICAL: Always include
DATE:   [Current Date]
SUBJECT: [Clear, Specific Topic]
CC:     [Other Recipients if applicable]
```

## Key Issues/Introduction

Write 1-2 paragraphs:
- State central problem requiring decision
- List questions management needs answered
- State required goals (numbered) that solutions must achieve

**Rules:**
- Goals listed here become decision criteria
- Do NOT mention recommendations
- Do NOT repeat known background information

## Recommendations

Format as numbered bullet points:
1. [First recommendation]
2. [Second recommendation]
3. [Third recommendation]

**Rule:** No rationale here—that goes in Analysis.

## Decision Criteria

Short paragraph explaining:
- How criteria derive from stated goals
- Weight rationale (1-5 scale)
- How criteria measure goal achievement

## Analysis

For each issue:
1. State the issue and alternatives (short paragraph)
2. Analyze each alternative's pros/cons
3. Reference decision matrix exhibit
4. State recommended alternative with reasoning

## Conclusion

2-3 paragraphs summarizing:
- Key recommendations
- Implementation priorities
- Expected outcomes

**Rules:**
- Do NOT make predictions outside management's control
- Do NOT introduce new information
