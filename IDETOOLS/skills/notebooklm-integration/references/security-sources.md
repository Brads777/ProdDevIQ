# Security Sources by Tech Stack

Curated URLs for building a security knowledge base with NotebookLM. Add these as sources to your `security-handbook` notebook.

---

## Universal (All Projects)

```bash
nlm source add security-handbook --url "https://owasp.org/www-project-top-ten/" -w
nlm source add security-handbook --url "https://cheatsheetseries.owasp.org/" -w
nlm source add security-handbook --url "https://owasp.org/www-project-web-security-testing-guide/" -w
nlm source add security-handbook --url "https://cwe.mitre.org/top25/archive/2024/2024_cwe_top25.html" -w
```
